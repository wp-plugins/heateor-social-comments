Integrate Facebook Comments, Google Plus Comments and/or Disqus Comments on your website

== Description ==
This plugin let you transform the commenting system of your website. You can enable Facebook Comments, Google Plus Comments, Disqus Comments and/or default WordPress comments simultaneously. You have full control over which of these four commenting systems you want to enable. You can choose between stacked and tabbed interface with re-ordering of the comments.

= Features: =
1. Enable **Facebook Comments**
2. Enable **Google Plus Comments**
3. Enable **Disqus Comments**
4. Choose between **Tabbed and Stacked layouts**
5. **Rearrange order** of enabled commenting systems
6. **Customize labels** for individual comment system as well as combined comment system
7. **Show/Hide comment counts** for individual comment system
8. Options to **enable Social Commenting on default post types (posts, pages) as well as custom post types** which support comments
9. Enable **Social Comments with/without default comments system**
10. **Customizable behavior for individual comment system**, like - Target Url, language, width of commenting interface, label etc.

== Installation ==

= Installation =
* Download the plugin from this page
* Navigate to **Plugins** > **Add New** section in your website admin panel
* Click **Upload**, click **Choose file**, upload the plugin zip and click **Install Now** button
* Activate the plugin when installation is complete

= Configuration =

After installing and activating the plugin, you will see **Social Comments** menu in the left sidebar in your website admin panel.
You can configure required options there.
There is **help** available with each option. You can see the help text by clicking the **Question Mark** icon before each option.

= Shortcode =

You can use **[Heateor-SC]** Shortcode in the content of required page/post where you want to display Social Comments.

Optionally, you can use **style** attribute in the Shortcode to style the rendered commenting interface.

For example

**[Heateor-SC style="background-color:#000;"]**
